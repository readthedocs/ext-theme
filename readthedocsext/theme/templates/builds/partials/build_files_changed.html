{% comment "rst" %} Build files changed =================== ``builds/partials/build_files_changed.html`` Renders the contents of the "Files changed" tab on the build detail page. It shows the file tree diff for a build as a single, flat list of the documentation pages that were added, modified, or deleted compared to a base. Pull request builds are compared against the base version, while normal version builds are compared against the version's previous build.
