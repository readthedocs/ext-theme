{% comment "rst" %} Build files changed =================== ``builds/partials/build_files_changed.html`` Renders the contents of the "Files changed" tab on the build detail page. It shows the file tree diff for a build: the documentation pages that were added, modified, or deleted compared to a base. Pull request builds are compared against the base version, while normal version builds are compared against the version's previous build. Template parameters ------------------- .. describe:: build The build instance the diff belongs to. .. describe:: files_changed_diff A ``FileTreeDiff`` instance with the added, modified, and deleted files. {% endcomment %} {% load i18n %}
